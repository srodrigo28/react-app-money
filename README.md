# Primeiro Projeto com React-App

* comando para iniciar um projeto
* Usando yarn
```
yarn create-react-app app-money --template typescript
```

* Usando npm
```
npx create-react-app app-money --template typescript
```

* entrar na basta
cd app-money

* iniciar a o projeto
